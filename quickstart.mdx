Getting started with thanks.io is easy!  Follow the steps below to create an account, get an API key, and submit your first order.

<Steps>
  <Step title="Create Account">
    Create an accounts at [thanks.io](https://thanks.io) if you haven't already.
  </Step>
  <Step title="Create An API Key">
    Visit https://dashboard.thanks.io/profile/api to create an API key.
  </Step>
  <Step title="Enable Testing Mode">
    Turn on API/Zapier Test Mode to avoid sending real mail while testing.  See [API Mode](api-mode) for more details.
  </Step>
  <Step title="Submit Your First Order">
  ```bash
  curl -X POST "https://api.thanks.io/api/v2/send/postcard" \
    -H "Content-Type: application/json" \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -d '{
      "recipients": [{ 
        "name": "John Doe", 
        "address": "123 Main Street, Any Town, NY 12345"
      }],
      "message": "Hi %FIRST_NAME% - Thank you for your business!",
      "front_image_url": "https://d2md0c8rpvzmz5.cloudfront.net/inspiration_templates/default10.png"
    }'
    ```
  </Step>
  <Step title="Check Your Order Status">
    You can check the status of your order by visiting the [Order History Page](https://dashboard.thanks.io/orders_history).
  </Step>

  
</Steps>
